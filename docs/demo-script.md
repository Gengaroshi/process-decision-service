## Demo (2–3 min)

1) POST /api/jobs/evaluate
   - pokaż decyzję (plan akcji) + zapis w DB

2) POST /api/jobs/{id}/execute
   - pokaż wykonanie akcji integracyjnych (mock) i status

3) GET /api/jobs/{id}
   - pokaż audyt + pełny stan

4) GET /api/metrics
   - pokaż proste KPI (ile jobów, ile w statusach)
